Congratulations on choosing the HAL-42.

The latest calculator by FCC

Equiped with the most advanced functionalities:

  • Percent calculation: For example, to calculate 10% off £120, simply type 1 2 0 - 1 0 % =
  • Easy squaring: For example, to calculate the square of 234, simply type 2 3 4 × ×
  • Compound interests: For example, to see how much you'll have in an account with a 5% interest when you deposit £2100, simply type 1 . 0 5 × × =. Ignore the result and type 2 1 0 0 =. Pressing = again gives you the interest for each subsequent year.
  • And many more!
